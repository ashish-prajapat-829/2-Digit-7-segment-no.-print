🔢 Arduino + Proteus 7-Segment Display Simulation

A virtual simulation project built in Proteus to display numeric digits on a 7-segment display using Arduino.

🚀 Project Overview

This project demonstrates the interfacing of a 7-segment display with Arduino in Proteus simulation software.
It helps beginners understand how digital outputs control display segments in embedded systems.

The project is fully simulated in Proteus before real hardware implementation.

✨ Features

✅ Arduino-based 7-segment display control
✅ Proteus virtual simulation
✅ Displays numbers using GPIO pins
✅ Beginner-friendly embedded project
✅ Supports single and multi-digit display logic
✅ Easy to modify for timers, clocks, and counters

⚡ Working Principle

The Arduino sends digital HIGH and LOW signals to the segments (a–g) of the display.

Different segment combinations generate different numeric digits.

🔌 Segment Connections

a	          12
b           11
c         	10

e           08
f          	07
g         	06
